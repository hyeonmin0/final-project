# 기간별 환율 변화 시각화 프로그램

## 프로젝트 개요
이 프로그램은 실시간으로 사용자가 입력한 기준 통화와 대상 통화에 대한 환율 변화를 조회하고, 조회된 환율 데이터를 시각적으로 그래프로 나타내는 프로그램입니다. 사용자가 입력한 날짜 범위에 대해 최저, 최대, 평균 환율과 환율 변화량을 계산하여 제공하며, 시각화를 위한 그래프도 제공합니다.

## 사용된 라이브러리
-**requests** : 환율 데이터를 가져오기 위해 Frankfurter API와 통신할 때 사용합니다.

-**datetime** : 날짜 형식을 처리하고 유효성을 검사하는 데 사용합니다.

-**math** : 환율 변동량과 변동 비율 계산을 위해 사용합니다.

-**matplotlib** : 환율 변화를 시각화하는 그래프를 그리기 위해 사용합니다.

## 코드 구성
### 1. API를 통한 환율 데이터 가져오기
-사용자가 **기준 통화**와 **대상 통화**를 입력하면, 해당 통화의 환율을 **Frankfurter API**에서 실--시간으로 가져옵니다.

### 2.통계 정보 제공
환율의 **최저**, **최고**, **평균** 등을 계산하여 출력합니다.
-**변동량**과 **변동 비율**을 계산하여 알려줍니다.

### 3. 환율 변화 시각화
**Matplotlib**를 사용하여 날짜별 환율 변화를 그래프로 시각화합니다.
-이 그래프는 환율 변동 추이를 직관적으로 보여줍니다.

### 4. 날짜 유효성 검사
-사용자가 입력한 날짜 형식이 정확한지 **datetime**을 통해 확인하고, 잘못된 날짜 형식에 대한 **오류 메시지**를 출력합니다.
